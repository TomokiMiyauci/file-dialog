File Select Dialog

JavaScriptからファイル選択ダイアログを直接呼び出す

はじめる

💡 Promiseベース

ファイルの選択を待ち受けるPromiseを返します。また、複数のファイルの選択とファイルタイプの制限を受け付けます。

⚡️ とても小さなバンドルサイズ

依存関係はなく、最適化されESモジュール形式で配布されています。

🛠️ 純粋なTypescript

APIには完全なTypeScriptタイピングを備えています。
